# 🌍 gen-231

> مولّد حسابات وسائل التواصل الاجتماعي مع IP عالمي حقيقي
> Social Accounts Generator with Real Worldwide IP

[![Streamlit](https://img.shields.io/badge/Streamlit-FF4B4B?logo=streamlit&logoColor=white)](https://streamlit.io)
[![Python](https://img.shields.io/badge/Python-3.8+-blue?logo=python&logoColor=white)](https://python.org)
[![License](https://img.shields.io/badge/License-MIT-green.svg)](LICENSE)

## ✨ المميزات

- 🌍 دعم **231 دولة** حول العالم
- 📡 **6,778+ نطاق IP** حقيقي من سجلات IANA/RIR
- 💼 **12 مهنة** احترافية مع تخصصات
- 🎓 **60+ جامعة** حقيقية حول العالم
- 📝 **بايو احترافي** جاهز للنسخ المباشر
- 🌐 دعم كامل للعربية والإنجليزية
- 🎨 واجهة RTL بخط Noto Sans Arabic

## 📦 محتويات المشروع

| الملف | الوصف |
|---|---|
| `streamlit_app.py` | الواجهة الرئيسية |
| `profession_data.py` | وحدة المهن والتعليم |
| `ip_engine.py` | محرك توليد IP |
| `countries_data.py` | بيانات 250 دولة |
| `ip_ranges_compact.json` | قاعدة بيانات IP الرسمية |
| `requirements.txt` | متطلبات Python |

## 🚀 التشغيل المحلي

```bash
git clone https://github.com/YOUR_USERNAME/gen-231.git
cd gen-231
pip install -r requirements.txt
streamlit run streamlit_app.py
```

## ☁️ النشر على Streamlit Cloud

1. ارفع الملفات إلى GitHub
2. اذهب إلى [share.streamlit.io](https://share.streamlit.io)
3. اربط الـ Repository
4. حدد `streamlit_app.py` كملف رئيسي
5. اضغط **Deploy** 🚀
